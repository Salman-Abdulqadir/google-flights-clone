## 🚀 Features

- 🌍 Multi-city flight search capabilities
- 📅 Flexible date picker with price calendar
- ⚡ Real-time price comparisons
- 🔍 Advanced filters (stops, airlines, price range)
- 📱 Fully responsive design
- ✨ Interactive flight path visualization

## 🛠️ Tech Stack

- **Frontend**:
  - ⚛️ React 18
  - 🎨 Material UI 5
  - 🛣️ React Router 6
- **Backend**:
  - 🔄 TanStack Query v4
  - 📡 Axios
- **Tooling**:
  - ⚡ Vite
  - 🧶 Yarn
  - 🔑 Environment Configuration

## 🚨 Getting Started

### Prerequisites

- Node.js ≥16.0
- Yarn

### Installation

1. Clone the repository:

bash
git clone https://github.com/yourusername/sky-search.git

2. Install dependencies:

```bash
yarn install
```

3. Set up environment variables:

```bash
cp .env.example .env
```

4. Start development server:

```bash
yarn dev
```
